# portfolio_tutorial
Meu portfólio pessoal que pode ajudar você a criar o seu!

**No momento ele está em processo de construção. As etapas, status e futuras tasks estão em issues e millestones, caso queira acompanhar enquanto cria o seu!**

Esse repositório usa tecnologias como: Node.js, Next.js e React.
Tem o intuito de apresentar meu trabalho e experiências, além de auxiliar no desenvolvimento de portfólios de desenvolvedores iniciantes ou que queiram aprender um pouco sobre as tecnologias usadas, hospedagem e deploys.
Nós vamos aprender juntos com base nos ensinamentos do @filipedeschamps no curso.dev, recomendo assinar, vale muito a pena! 😉

Esse readme é provisório, logo será estilizado e abastecido com melhores informações!
